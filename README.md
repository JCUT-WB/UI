# wubo-ui

### 快速开始

### 1、安装组件库
```bash
    npm i wubo-ui
```
### 2、引用组件库
```javascript
    //全部引入
    import 'wubo-ui/dist/css/index.css';
    import MUI from 'wubo-ui';
    VUe.use(MUI);

    //按需引用
    import 'wubo-ui/dist/css/index.css';
    import { Demo } from 'wubo-ui';
    Vue.use(Demo);
```
